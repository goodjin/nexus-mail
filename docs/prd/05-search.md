# 模块 PRD：邮件搜索（US-005）

## 目标
提供简洁且高效的搜索入口，默认聚焦当前文件夹，避免额外 UI 干扰。

## 范围
**包含**
1. 列表顶部搜索框
2. 默认当前文件夹范围
3. 实时结果更新与空状态

**不包含**
- 搜索范围切换 UI（v1 不展示）
- 高级过滤器面板（v1 不展示）

## 搜索体验
1. 列表顶部仅保留搜索输入框
2. 输入即触发搜索结果刷新（可做轻量 debounce）
3. 清空输入恢复当前文件夹原列表
4. 搜索结果仅替换列表内容，不改变详情面板
5. 搜索范围字段：主题、发件人、收件人、正文

## 搜索范围
- 默认：当前选中文件夹
- 文件夹切换时：自动更新搜索上下文
- 不在 UI 中展示范围选择器、过滤器栏、搜索历史或结果摘要

## 空状态与错误
- 无结果：显示“未找到匹配邮件”
- 加载失败：显示错误原因并支持重试

## 验收标准（AC）
- AC-005-01: 搜索响应时间 < 500ms（本地索引）
- AC-005-02: 支持按文件夹范围搜索
- AC-005-03: 搜索框默认在当前文件夹执行搜索，并随文件夹切换自动更新上下文
- AC-005-04: 无结果时显示友好提示
- AC-005-05: 邮件列表上方仅保留搜索输入，不展示范围切换、过滤器、搜索历史或状态摘要
- AC-005-06: 列表搜索框的清空、输入和结果刷新不影响当前邮件详情面板状态
