# 前端开发计划 - 邮件列表与详情

## 文档信息
- **模块编号**: MOD-03
- **模块名称**: 邮件列表与详情
- **优先级**: P0

## 1. 模块概述
实现邮件列表卡片、虚拟滚动、邮件选中、详情阅读、附件入口和 XSS 安全展示。

## 2. 页面清单

| 页面/组件 | 功能 |
| --- | --- |
| `EmailList` | 列表、搜索框、批量选择入口 |
| `EmailDetail` | 正文、附件、操作按钮 |
| `App` | 列表与详情联动 |

## 3. 开发任务拆分

| 任务编号 | 任务名称 | 文件数 | 行数 | 依赖 |
| --- | --- | --- | --- | --- |
| T-01 | 列表卡片信息完整化 | 2 | ~140 | - |
| T-02 | 详情头信息展示 | 2 | ~120 | T-01 |
| T-03 | 附件下载与预览入口 | 2 | ~180 | T-02 |
| T-04 | HTML 安全渲染加固 | 2 | ~120 | T-02 |
| T-05 | 列表-详情联动优化 | 2 | ~100 | T-01~04 |
| T-06 | 前端测试补齐 | 3 | ~180 | T-01~05 |

## 4. 详细任务定义

### T-01: 列表卡片信息完整化
- [ ] 列表项显示发件人、主题、摘要、时间、附件图标
- [ ] 未读样式符合 PRD

### T-02: 详情头信息展示
- [ ] 显示发件人、收件人、时间、主题
- [ ] 预留 headers 展开区

### T-03: 附件下载与预览入口
- [ ] 支持单文件下载
- [ ] 为图片/PDF 内联预览预留入口

### T-04: HTML 安全渲染加固
- [ ] `DOMPurify` 应用于详情正文
- [ ] 恶意脚本不可执行

### T-05: 列表-详情联动优化
- [ ] 切换邮件后详情刷新
- [ ] 删除当前邮件时选中状态正确回收

## 5. 测试策略
- 组件测试: EmailList / EmailDetail
- 安全测试: XSS 渲染
- E2E: 打开邮件 -> 查看详情 -> 下载附件
