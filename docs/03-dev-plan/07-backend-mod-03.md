# 后端开发计划 - 邮件列表与详情

## 文档信息
- **模块编号**: MOD-03
- **模块名称**: 邮件列表与详情
- **优先级**: P0

## 1. 模块概述
负责邮件列表查询、详情读取、正文净化前置数据、附件元数据和头信息读取能力。

## 2. 对应PRD

| PRD编号 | 功能 |
| --- | --- |
| FR-005 | 邮件列表展示 |
| FR-006 | 邮件详情查看 |
| FR-010 | 附件预览 |

## 3. 技术设计
- 目标文件:
  - `src-tauri/src/commands.rs`
  - `src-tauri/src/core/database.rs`
  - `src-tauri/src/core/imap_client.rs`

## 4. 开发任务拆分

| 任务编号 | 任务名称 | 文件数 | 行数 | 依赖 |
| --- | --- | --- | --- | --- |
| T-01 | 列表查询 DTO 整理 | 2 | ~120 | - |
| T-02 | 详情读取与头信息 | 3 | ~180 | T-01 |
| T-03 | 附件元数据与下载接口 | 3 | ~160 | T-02 |
| T-04 | 详情缓存策略 | 2 | ~100 | T-02 |
| T-05 | 后端测试补齐 | 4 | ~180 | T-01~04 |

## 5. 详细任务定义

### T-01: 列表查询 DTO 整理
- [ ] 列表摘要字段覆盖发件人、主题、摘要、时间、附件标识
- [ ] 支持按文件夹分页读取

### T-02: 详情读取与头信息
- [ ] 支持 HTML / Text 双正文
- [ ] 支持 headers 展开数据

### T-03: 附件元数据与下载接口
- [ ] 元数据字段与 PRD `Attachment` 对齐
- [ ] 下载接口支持单附件获取

### T-04: 详情缓存策略
- [ ] 列表切详情后不重复请求同一封邮件
- [ ] 缓存失效规则明确

## 6. 测试策略
- 列表查询测试
- 详情正文测试
- 附件下载测试
