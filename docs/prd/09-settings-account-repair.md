# 模块 PRD：设置中心与账号修复（US-008）

## 目标
集中管理应用偏好、同步与隐私设置，并提供可操作的账号修复流程。

## 范围
**包含**
1. 设置入口与分区
2. 主题、删除确认、下载目录、远程图片策略
3. 账号状态、同步信息与修复入口

## 设置分区
1. **通用**
   - 主题模式（浅色/深色/跟随系统）
   - 删除前确认开关
   - 搜索历史条数上限
2. **账户**
   - 账户列表、状态、最近错误
   - 修改密码 / 重新认证 / 编辑服务器配置
3. **同步**
   - 上次同步时间
   - 手动刷新
   - 同步启用/停用、同步间隔
4. **隐私与附件**
   - 远程图片加载策略
   - 默认附件下载目录

## 账号修复流程
1. 账户出现异常时，在设置页标红并显示错误文案
2. 用户点击“修复”进入修复流程：
   - 修改密码
   - 重新 OAuth2 授权
   - 编辑 IMAP/SMTP 配置
3. 修复完成后自动测试连接并同步

## 错误提示
- 显示最近一次错误原因
- 提供可执行的“下一步操作”

## 验收标准（AC）
- AC-008-01: 支持浅色/深色/跟随系统切换
- AC-008-02: 支持配置删除前确认对话框开关
- AC-008-03: 支持配置远程图片加载策略和默认附件下载目录
- AC-008-04: 账号异常时可直接从设置中心进入修复流程
- AC-008-05: 设置中心可查看上次同步时间并触发手动刷新
