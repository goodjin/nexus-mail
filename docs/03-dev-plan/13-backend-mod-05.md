# 后端开发计划 - 搜索与组织管理

## 文档信息
- **模块编号**: MOD-05
- **模块名称**: 搜索与组织管理
- **优先级**: P1

## 1. 模块概述
负责全文搜索、过滤范围、星标/已读/删除/移动/归档等组织管理命令和索引维护。

## 5. 详细任务定义

### T-01: 搜索索引字段扩展
- [ ] 覆盖主题、发件人、正文
- [ ] 为按文件夹范围搜索预留条件

### T-02: 搜索过滤器接口
- [ ] 支持发件人/日期/附件条件
- [ ] 不跨账户泄漏

### T-03: 搜索历史存储
- [ ] 保留最近 10 条
- [ ] 可清理/覆盖

### T-04: 标记/删除/归档统一命令
- [ ] 已读、未读、旗标、删除统一错误模型
- [ ] 便于前端批量调用

### T-05: 移动邮件命令设计
- [ ] 为拖拽移动提供 folder target 参数
- [ ] 同步本地与远端语义
