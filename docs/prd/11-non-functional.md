# 非功能需求（技术/性能/安全/跨平台）

## 技术架构
- 框架：Tauri v2.x
- 后端：Rust（tokio）
- 前端：Web（React）
- IPC：Tauri Command（JSON）

## 协议支持
| 协议 | 支持 | 说明 |
|-----|------|------|
| IMAP4 | P0 | 邮件同步 |
| SMTP | P0 | 邮件发送 |
| POP3 | P1 | 可选支持 |
| EWS | P2 | Exchange 邮件 |

## 安全与加密
- IMAP/SMTP：TLS 1.2+
- 本地存储：敏感信息加密（AES-256）
- 凭据：系统密钥链集成，不落地明文

## 性能指标
| 指标 | 要求 | 说明 |
|-----|------|------|
| 冷启动 | < 2 秒 |  |
| 邮件列表加载 | < 500ms | 1000 封邮件 |
| 搜索响应 | < 500ms | 本地索引 |
| 内存占用 | < 200MB | 正常使用 |
| 安装包大小 | < 30MB | Windows |

## 跨平台支持
| 平台 | 支持 | 优先级 |
|-----|------|--------|
| Windows 10+ | P0 | 高 |
| macOS 11+ | P0 | 高 |
| Linux | P1 | 中 |
| iOS | P2 | 低 |
| Android | P2 | 低 |
