# 集成测试计划 - 撰写、发送与回复转发

## 文档信息
- **模块编号**: MOD-04
- **模块名称**: 撰写、发送与回复转发
- **优先级**: P0

## 1. 测试场景

| 场景编号 | 场景名称 | 预期结果 |
| --- | --- | --- |
| INT-001 | 新建邮件发送成功 | 邮件发送并关闭弹窗 |
| INT-002 | 附件发送 | 附件随邮件发出 |
| INT-003 | 回复邮件 | 自动带上原主题/正文 |
| INT-004 | 发送失败 | 内容保留且提示错误 |
| INT-005 | 自动保存草稿 | 草稿状态可恢复 |
