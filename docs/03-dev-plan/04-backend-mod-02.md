## 6. 详细任务定义

### T-01: 文件夹映射规则整理
- **要求**: 明确 INBOX/SENT/DRAFTS/SPAM/TRASH/ARCHIVE 的远程映射和本地回退策略。
- **验收标准**:
  - [ ] 支持系统文件夹优先排序
  - [ ] 未知文件夹保持原样

### T-02: 增量同步算法补完
- **要求**: 统一 last UID / first UID / 远端删除策略。
- **验收标准**:
  - [ ] 新邮件可增量拉取
  - [ ] 已删除邮件可裁剪
  - [ ] 不重复写入邮件

### T-03: 历史邮件分页接口
- **要求**: 为滚动加载提供稳定 offset/批次协议。
- **验收标准**:
  - [ ] 支持 50 条为一批的读取
  - [ ] 接口返回顺序稳定

### T-04: 未读数更新统一化
- **要求**: 列表、详情、文件夹 badge 使用同一未读统计来源。
- **验收标准**:
  - [ ] 已读/未读切换后文件夹计数正确

### T-05: 删除/裁剪一致性
- **要求**: 本地删除、远端删除、同步裁剪保持语义一致。
- **验收标准**:
  - [ ] 不产生孤儿附件数据
  - [ ] 不误删其他文件夹邮件
